Market Data Strategy — Three Options

The strategies are built on daily bars — only the price trigger check needs real-time data

Key insight: RSI, ATR, SMA are computed from historical daily closes — those don't change during the day and need no real-time feed. Only the trigger price check ("is AAPL currently below $185?") needs a live price. This decouples the problem neatly.
A

All yfinance · 15-min delay everywhere

INDICATORS (daily)
yfinance daily bars — no delay issue, uses yesterday's close
PRICE TRIGGER
yfinance fast_info — ~15 min delay on current price

Pros

  • Zero additional setup or accounts
  • One data source, simple
  • Acceptable for daily swing trades — signal at 10:02am on 9:47am price is fine

Cons

  • Can miss intraday entry windows if price moves fast
  • Not suitable if you want to catch breakouts in real time
Cost: free  |  Setup: none  |  Recommended for: MVP
B

yfinance history + Alpaca IEX WebSocket · near real-time triggers

INDICATORS (daily)
yfinance daily bars — same as always, no change
PRICE TRIGGER
Alpaca IEX WebSocket — near real-time trades, max 30 symbols simultaneously

Pros

  • Near real-time price for watchlist candidates
  • Free Alpaca account (paper trading, no real money)
  • 30-symbol limit is fine — morning scan narrows candidates first
  • WebSocket is push-based, no polling needed

Cons

  • IEX = subset of trades (~10-15% of volume) — still representative for liquid large-caps
  • Requires Alpaca account + API key setup
  • 30-symbol limit means you must triage candidates each morning
Cost: free  |  Setup: Alpaca account + API key  |  Recommended for: production
C

yfinance history + Polygon.io · true real-time, all symbols

INDICATORS (daily)
yfinance daily bars — same as always, no change
PRICE TRIGGER
Polygon.io WebSocket — true consolidated tape, unlimited symbols

Pros

  • True real-time SIP (consolidated tape) — all exchanges
  • No symbol limit on streaming
  • Excellent API, well-documented Python SDK
  • Can also replace yfinance for historical if desired (one source)

Cons

  • $29/month minimum for real-time (free tier is 15-min delayed)
  • More than needed for a signal-only daily strategy
Cost: $29/mo  |  Setup: account + API key  |  Recommended for: if budget allows
Recommended path: Start with A (all yfinance) to build and validate the full pipeline. Upgrade to B (Alpaca IEX) once everything works — it's free and the only change is the price-trigger data source. Upgrade to C only if you find the 30-symbol or IEX-coverage limits actually causing missed signals.
